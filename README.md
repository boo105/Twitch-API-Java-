# Twitch-API-Java-

학교 자바프로그래밍 기말 프로젝트로 만듬.
 
자바 Swing를 사용해 GUI 구현

트위치에서 제공하는 API 이용해

사용자의 트위치 ID를 입력하면<br/> 
사용자가 follow 한 스트리머 이름, 기타정보 와 생방송 유무 들을 출력

특정 스트리머를 클릭하게 되면<br/>
생방송인 경우 실시간 채팅을 할수있게 채팅창을 띄우고<br/>
오프라인 인 경우는 클립 모음 사이트를 띄어줌


원래는 생방송인 경우 생방송을 틀어주려고 했으나<br/>
webView 에서 실시간 스트리밍을 안나오는 문제가 생겨서<br/>
실시간 채팅을 띄어주게 됬음.
